## Introduction

Agentic scientist with a crazy imagination hooking right into your coding agent to work as your lab partner.

- **Micro Papers.** Writes paper-like prose to outline the experiment, findings and conclusions.
- **Research-Grounded.** Collects evidence, research and literature to back up claims and knowledge.
- **Experiments & Benchmarks.** Runs experiments to verify correctness or disprove theory.
- **Pull Request Agnostic.** Writes outcomes and code into a pull request for peers to review.

## Getting Started

### Prerequisites

- [Python 3.12+](https://www.python.org/downloads/)
- [uv](https://docs.astral.sh/uv/)

### Installation

```sh
uv sync
```

## Development

Experiments are test-first — outline your research as tests, then run:

```sh
uv run test
```
